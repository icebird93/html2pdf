# Electronic Signature 

[back](./README.md)

You can add an electronic signature to the PDF, by using the following specific html TAG:

```
<cert
    src="/path/to/cert.crt"
    name="sender_name"
    location="sender_location"
    reason="sender_reason"
    contactinfo="sender_contact"
>                            
/** html **/
</cert>
```
